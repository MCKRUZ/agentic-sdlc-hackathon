The Agent Loop

Every agent action follows this cycle
Step 01
Observe
Read context: files, conversation history, your message. Build a picture of where things stand.
→
Step 02
Plan
Decide what steps to take. Break the task down. Sequence the actions before touching anything.
→
Step 03
Act
Execute: write, run, read, ask. Each action is visible before it lands.
→
Step 04
Verify
Check the result. Update understanding. If something didn't work, loop back and try a different approach.
You are in the loop. Every action is visible and reversible before you accept it.
