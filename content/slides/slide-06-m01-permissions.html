Module 01 — The Agentic Mental Shift
Permission model

The Agent Asks Before It Acts

Green — Automatic
No confirmation needed
  • Read files and directories
  • Search the codebase
  • Explain code
  • Run tests (when you've approved)
Yellow — Asks First
Requires your go-ahead
  • Create or modify files
  • Run shell commands
  • Install packages
  • Make multi-file changes
Red — Always Confirms
Explicit confirmation, always
  • Delete files or branches
  • Force-push to git
  • Modify production config
  • Run destructive operations
You set the permission level. The tool doesn't act autonomously unless you tell it to.
