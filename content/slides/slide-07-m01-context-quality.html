Module 01 — The Agentic Mental Shift
Context quality

Garbage In, Generic Out

Vague Prompt
"Fix the bug"
Result
Guesses. Edits the wrong thing. Asks 5 clarifying questions before doing anything useful.
Context-Rich Prompt
"The validateUser() function in src/auth/validators.ts is returning false for valid OAuth tokens. The token format changed in v2.3 — it now includes a 'sub' field instead of 'userId'. Fix the validation logic."
Result
Reads the file. Understands the change. Makes the targeted fix — first time.
The quality of the agent's output is directly proportional to the quality of context you give it.
