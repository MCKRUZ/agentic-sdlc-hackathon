Module 02 — Out-of-the-Box Usage
Zero-config workflow

Open a Project. Start the Tool. Ask a Question.

1
Navigate to your project
Launch the agent
$ cd my-project
$ claude
# That's it. You're in.
2
Ask a navigation question
Orient the agent
"What are the main entry points for this application?"
Ground it in your actual structure before asking it to change anything.
3
Review the grounded response
Trust what's cited
It reads actual files. It cites actual paths. No hallucination — just your code, reflected back with context.
No config. No context file. No setup beyond install. This is the baseline.
