Module 02 — Out-of-the-Box Usage
Agent context scope

The Agent Reads Your Actual Code

What it reads on first launch
In scope
  • ✓ Directory structure
  • ✓ File contents (on request)
  • ✓ Git history
  • ✓ Package files (package.json, .csproj, etc.)
  • ✓ README and docs
  • ✓ Your conversation history in this session
What it does NOT automatically read
Out of scope
  • × Files you didn't ask about
  • × External systems
  • × Other projects
  • × Previous sessions (context resets each time)
  • × Your mind
Every file the agent reads consumes tokens. It is selective by default — and that's intentional.
