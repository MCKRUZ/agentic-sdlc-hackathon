LAB 01

Your First Agent
Interaction

30 minutes  ·  Use your own project
or the DevBoard sample
No configuration needed.
Just open your project and start typing.
What You'll Do
  • Open your project in the terminal and start the agent
  • Ask a navigation question — "What are the main entry points?"
  • Ask an explanation question — "Explain the most complex function"
  • Request a small change — add a comment block to one function
  • Review the diff before accepting
  • Run the project or tests to verify nothing broke
Goal: See what the tool can do with zero configuration. Don't configure anything yet.
