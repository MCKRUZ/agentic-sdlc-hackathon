Module 03 — Project Context Files

Context Is Layered

Tier 1
Global
~/.claude/CLAUDE.md
Applies to all your projects
Example: personal coding preferences, preferred tools
Tier 2
Project
[project-root]/CLAUDE.md
Applies to this repo — committed to git
Example: stack, conventions, what not to do
Tier 3
Folder
[subfolder]/CLAUDE.md
Applies only to that directory
Example: "this folder is auto-generated, don't edit"
Specificity — lower overrides broader
