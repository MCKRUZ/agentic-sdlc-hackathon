LAB 02

Write Your
Context File

40 minutes  ·  Your project gets a
CLAUDE.md today
The file you write today becomes part
of your project permanently.
What You'll Do
  • Create CLAUDE.md in your project root
  • Add: project name, tech stack with versions
  • Add: at least 3 coding conventions specific to your project
  • Add: at least 1 "never do this" rule
  • Run a prompt without the context file — note the response
  • Re-add the file and run the same prompt — compare the difference
  • Commit: git add CLAUDE.md && git commit -m "docs: add agent context file"
Goal: See the before/after difference. The first version doesn't need to be perfect.
