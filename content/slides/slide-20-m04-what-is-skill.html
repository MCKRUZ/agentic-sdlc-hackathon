Module 04 — Skills & Custom Commands

A Skill Is a Reusable Prompt Stored as a File

A skill is a markdown file in .claude/commands/ that you invoke by name. When you type /commit-message, the agent reads that file and executes its instructions against your current project state.
Step 01
✎
Write
.claude/commands/commit-message.md
Create the skill file with step-by-step instructions for the agent
›
Step 02
▶

Type the skill name in your agent session to run it instantly
