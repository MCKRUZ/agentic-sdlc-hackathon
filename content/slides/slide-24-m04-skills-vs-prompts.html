Module 04 — Skills & Custom Commands

When to Write a Skill vs. Just Prompt

Inline Prompt
Use when...
  • – It's a one-off task
  • – The context is highly specific to this moment
  • – You're still figuring out what you want
  • – You only need it once
vs
Write a Skill
Use when...
  • ✓ You've written this prompt more than once
  • ✓ Multiple teammates would benefit
  • ✓ The task has a consistent structure
  • ✓ Output format must be exact every time
  • ✓ You want it on a keyboard shortcut
The test: Could someone else on your team use this? If yes, it's a skill.
