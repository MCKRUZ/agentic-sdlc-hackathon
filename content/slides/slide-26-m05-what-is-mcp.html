Module 05 — MCP Servers

The Agent's Ability to Look Things Up

Model Context Protocol is a standard that lets your agent connect to external servers — to fetch live data, query databases, search documentation, or call APIs.

You register a server once. It's available in every conversation. The agent tells you when it's calling an MCP server and what it retrieved.

Key Distinction
MCP doesn't add AI capability. It adds current, accurate data. The agent's knowledge cutoff was months ago. Your libraries have been updated since then.
