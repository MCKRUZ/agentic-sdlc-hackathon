Module 05 — MCP Servers

The Servers Worth Knowing

context7
📚
What it does
Fetches up-to-date docs for any library your project uses
Install
claude mcp add context7 -- npx -y @upstash/context7-mcp@latest
filesystem
🗂
What it does
Gives the agent access to paths outside the project directory
Install
claude mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem /path
github
⚡
What it does
Query your GitHub data — issues, PR comments, commit history
Requires
GITHUB_TOKEN
postgres
🗄
What it does
Run read queries against your database from within a conversation
Install
claude mcp add postgres -- npx -y @modelcontextprotocol/server-postgres postgresql://...
brave-search
🔍
What it does
Search the web for current information during a conversation
Requires
BRAVE_API_KEY
