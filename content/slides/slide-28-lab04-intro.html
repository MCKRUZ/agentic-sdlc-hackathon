LAB 04

Add an MCP Server

30 minutes  ·  context7 recommended — or choose your own

What You'll Do
Install context7 claude mcp add context7 -- npx -y @upstash/context7-mcp@latest
Restart the agent /exit   then   claude
Verify the connection — /mcp should show context7 as connected
Ask a documentation question for a library your project uses Using the context7 docs, how do I configure connection pooling with better-sqlite3? Show the exact API.

Using the context7 docs, show me the correct way to handle middleware ordering in Express 5.
Compare the answer against the library's official docs — is it accurate?
Ask a second question for a different library or feature
Why This Matters
The agent's training data has a cutoff date. Library APIs change. MCP servers bridge that gap by fetching documentation at the moment you ask.
Goal
See the difference between training-data answers and live-doc answers. The gap is real.
After This Lab
You'll have a working MCP server registered in your Claude Code installation, available in every future conversation.
