Module 06 — Token Management

Tokens Are the Unit of Thought

Part 01
What a Token Is
A token is a chunk of text — roughly 3-4 characters or about three-quarters of a word.

Every message you send, every file the agent reads, every response it generates — all of it is measured in tokens and costs money.

More importantly: tokens are finite. Each model has a maximum context window.
Part 02
The Context Window
Your conversation + files read so far
~60% full
Token budget used in this session

When it fills up, the agent can't see earlier parts of the conversation. Quality drops.
Part 03
What Fills It Fastest
Large files read in full A 5,000-line file costs as much as hundreds of turns of conversation
Long conversation history Every turn is in the window — older turns don't disappear on their own
Re-reading the same files repeatedly Each read adds more tokens, even if the file hasn't changed
