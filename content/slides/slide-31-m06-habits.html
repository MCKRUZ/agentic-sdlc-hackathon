Module 06 — Token Management

Four Habits That Keep Conversations Sharp

1
Compact When Done
/compact
Summarizes the conversation and clears old context. Use it when a task is done but you're continuing in the same session.
The agent retains the summary — not every message. Same result quality, much lower token burn.
2
Start Fresh for New Tasks
/clear  or open a new session
Dragging old context into new work costs tokens and degrades quality. A new conversation is free.
The instinct to keep context alive is wrong. Each task deserves a clean slate.
3
Read Specifically, Not Broadly
Read the entire auth module → Read the validateUser function
The agent reads exactly what you ask for. Be precise. One function is almost always enough context.
Broad reads burn tokens on code that never influences the answer.
4
Put Permanent Context in Files
CLAUDE.md skill files reference docs
If you type the same thing into chat regularly, that text belongs in a file. Files load once — repeated chat context loads every turn.
CLAUDE.md is loaded automatically. Zero token cost per message, permanent context.
Symptom of context overload: the agent starts contradicting itself or forgetting earlier decisions. That's your signal to compact or start fresh.
