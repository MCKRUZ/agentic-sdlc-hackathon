Module 07 — Agentic SDLC

Don't Boil the Ocean. Start With One Thing.

1
This Week
Commit a context file to your real project
Not a sample project — the one you work in every day. A rough first version is fine. Refine it over the next two weeks.
You'll feel the difference within 48 hours. The agent will stop asking questions it should already know the answers to.
2
2
This Week
Build one skill for a task you do repeatedly
Look at your last 10 commit messages. That pattern is your first skill. If you write the same kind of code every week, automate the prompt.
One skill file per repeated task. Write the prompt once, reuse it indefinitely.
3
3
Next Sprint
Use the agent for one real task on your backlog
Not a tutorial. Not a toy example. A real task from your actual backlog. That's where you'll understand what it can and can't do.
The only way to know the ceiling is to hit it on work that actually matters.
